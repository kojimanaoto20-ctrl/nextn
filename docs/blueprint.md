# **App Name**: Ecosystem Risk Analyzer

## Core Features:

- Company Ecosystem Input Form: An interactive form for users to input detailed information about a company's industry, business model, core technology, ecosystem players, monetization, and risks.
- Strategy Score Calculation: Backend logic to process user inputs and calculate key metrics such as Platform Strength Score, Ecosystem Risk Score, and Value Capture Risk Score.
- Dashboard Visualization: Displays the calculated scores, a dynamic radar chart for ecosystem health, and an ecosystem map, providing a visual overview of the analysis.
- Strategic Memo Generation: Utilizes a generative AI tool to synthesize user inputs and calculated scores into a concise, actionable strategic memo, outlining key insights and recommendations.
- Framework Explorer Page: A dedicated section explaining the strategic frameworks from the Columbia Business School Technology Strategy course that underpin the analysis.
- Transparent Scoring Logic: Provides users with clear breakdowns and explanations for how each score is derived, ensuring trust and understanding of the analytical results.
- Mock Data Management: Enables loading and managing predefined mock company datasets (e.g., Uber, LinkedIn, Kakao) to showcase app functionality without real-time data input.

## Style Guidelines:

- Primary color: A sophisticated, deep sapphire blue (#2E4F8B) evoking trust, intelligence, and corporate strategy. This will be used for main accents, primary calls to action, and important headings.
- Background color: A very light, desaturated blue-gray (#F3F5F9) providing a clean, professional canvas that reduces eye strain and ensures content readability.
- Accent color: A vibrant teal (#2EBCAD) for highlights, interactive elements, and key data points, offering a refreshing contrast to the primary blue and indicating modern innovation.
- Headlines: 'Space Grotesk' (sans-serif) for its modern, tech-forward, and clean aesthetic, suitable for analytical displays and prominent titles.
- Body Text: 'Inter' (sans-serif) for its neutral, highly legible design, ideal for long-form content like strategic memos and framework explanations, ensuring comfortable reading.
- Utilize a consistent set of clean, minimalist line icons to represent data points, features, and navigation, maintaining a professional and consulting-style visual identity.
- Adopt a clean, grid-based layout with generous whitespace to emphasize readability and create a professional, consulting-report aesthetic. Prioritize responsive design for seamless viewing across devices.
- Implement subtle, purposeful animations for data transitions on charts and dashboard updates, enhancing user engagement without being distracting. Focus on smooth loading and state changes.
